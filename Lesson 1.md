# Start the MySQL database Learning

| Syntax        |  Explain      | example  |
| ------------- |:-------------:|:--------
| CREATE DATABASE        | 創建資料庫      | CREATE DATABASE `資料庫名稱` |
| USE        | 使用資料庫      | USE `資料庫名稱` |


#Create the Database
```mysql
-- CREATE DATABASE `資料庫名稱`
CREATE DATABASE `Company`;
```

#Use the Database
```mysql
-- USE`資料庫名稱`
USE `Company`;
```
